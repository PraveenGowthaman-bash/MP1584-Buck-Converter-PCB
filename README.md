 # MP1584 Buck Converter PCB

A compact DC-DC buck converter PCB designed using MP1584 switching regulator IC in Altium Designer.

## Features

- MP1584 based step-down regulator
- Compact PCB layout
- Power trace optimization
- Ground plane implementation
- Designed for manufacturability

## Tools Used

- Altium Designer
- Datasheet-based design methodology
